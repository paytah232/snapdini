# Snapdini Slideshow Music — Manifest

Royalty-free background music for the event-photo slideshow feature.

This manifest contains **two independent sets**:

1. **CC-BY 4.0 set** (`01..13`) — Kevin MacLeod / Incompetech. Commercial use OK
   but **attribution REQUIRED**. Documented in the original section below.
2. **No-attribution / CC0 set** (`c0-NN-…`) — see the dedicated section near the
   end of this file. Commercial use OK and **NO attribution required**. This set
   was added to let us avoid the mandatory CC-BY credit obligation.

There are currently **15** no-attribution tracks (Mixkit Free License), spanning
upbeat/party/groovy, warm "happy memories", and gentle/acoustic moods — enough
to **drop the CC-BY set entirely** if desired. See the no-attribution section
for the recommendation and the one license caveat to confirm first.

(CC-BY 4.0)** — https://creativecommons.org/licenses/by/4.0/

- **Commercial use: YES.** CC-BY 4.0 permits use in a commercial product and
  permits redistribution / bundling inside software.
- **Attribution: REQUIRED.** This is the only obligation. You must credit each
  track wherever users can reasonably find it (e.g. an in-app "Music credits" /

  exact attribution strings per track below.
- None of these are non-commercial-only or no-derivatives. All are safe for the
  Snapdini commercial use case.

### Caveat for the owner to review
- The original goal preferred **CC0 / public domain**. These tracks are **CC-BY,
  not CC0**, so the attribution obligation above is mandatory — there is no way
  to use them credit-free. If a fully attribution-free (CC0) set is required
  later, freepd.com (the obvious CC0 source) **permanently shut down in 2025**,
  and other CC0 sources (Pixabay Music, FMA CC0 filter) require browser/API
  flows that did not offer reliable `curl`-able direct URLs at fetch time.
  Pixabay Audio (CC0-like Pixabay Content License) is the best candidate for a
  human to grab additional credit-free tracks manually.

## Licensing summary — READ BEFORE SHIPPING (no-attribution set)

All 15 tracks below come from **Mixkit** (mixkit.co) and are licensed under the
**Mixkit Free Stock Music License** — https://mixkit.co/license/

- **Commercial use: YES.** The Mixkit Free License permits use in personal *and*
  commercial projects.
- **Attribution: NO — not required.** Mixkit explicitly states attribution is
  appreciated but not required, and no sign-up is required.
- **Bundling into a generated slideshow video: YES.** The intended Snapdini use
  (mixing a track into an auto-generated event slideshow video that we deliver
  to the customer) is an allowed "end product" use — the music is incorporated
  into a larger work, not redistributed on its own.

### Caveat for the owner to confirm before shipping
The Mixkit Free License has a few **media-format / standalone restrictions** that
do NOT affect the current slideshow use but should be confirmed against any
future plans:
- You may **not redistribute the tracks on a standalone basis** (i.e. don't ship
  the bare .mp3 files for users to download/extract; only deliver them embedded
  in the slideshow video).
- You may **not register/claim the tracks as your own** or remix-and-resell them
  as music.
- Mixkit's terms restrict use on **CDs/DVDs, video games, and TV/radio
  broadcast**. Snapdini slideshow video (web/app delivery) is fine; broadcast
  distribution would not be.
If Snapdini's delivery stays "track embedded inside a generated slideshow video
served to the event's guests," this set is safe and **credit-free**.

## Can we drop the CC-BY set entirely?
**Yes, technically** — the no-attribution set has 15 tracks covering all three
target moods, which is enough to power the slideshow feature credit-free. Before
deleting tracks 01..13, confirm the Mixkit standalone-redistribution caveat
above is satisfied by the delivery model (track embedded in generated video, not
shipped as a bare file). Keeping both sets is also fine; nothing requires the
CC-BY tracks to be removed.
